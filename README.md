# Zenn CLI

* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

👇  新しい記事を作成する
```
zenn new:article
```
👇  新しい本を作成する
```
zenn new:book
```
👇  投稿をプレビューする
```
zenn preview
```
